Place all product photos in this folder.

How to use with Excel:
- Put image files here, for example:
  - `mini-gt-nissan-s15-silvia.jpg`
  - `mclaren-750s-orange.jpg`
  - `oracle-red-bull-rb20.jpg`
- In `inventory.xlsx`, use the `image`, `image2`, and `image3` columns.
- Type only the filename if the file is in this folder.

Example:
- `image` = `mini-gt-nissan-s15-silvia.jpg`
- `image2` = `mini-gt-nissan-s15-silvia-back.jpg`

The storefront will automatically resolve those to:
- `images/mini-gt-nissan-s15-silvia.jpg`
- `images/mini-gt-nissan-s15-silvia-back.jpg`

Do not embed images inside Excel cells. The site reads text file paths, not pasted Excel pictures.
